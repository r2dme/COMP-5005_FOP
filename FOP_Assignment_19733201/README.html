File Descriptions


File 1 - shrimpSimBase_v2.py : main executable python program file.**Do not delete**


File 2 - shrimptest.py : A program file that contains classes to be executed in the main executable program **Do not delete**


File 3 - shrimp_sweep.sh : A program file which contains executable for parameter sweep for the simulation **Do not delete**


File 4 - Report.docx : The report for overall programming assignment


File 5 - User_Guide.pdf : The user guide to execute the simulation


File 6 - README


Dependencies


To execute the simulation, the following files should be present in the current directory

1. shrimpSimBase_v2.py

2. shrimptest.py

3. shrimp_sweep.sh




To Execute the Simulation

Command line Argument:

python3 shrimpSimBase_v2.py <number of shrimps> <simulation time steps>

Here input the required number of shrimps and simulation time to be simulated, in place of <number of shrimps> & <simulation time steps>


Parameter sweep:

sh shrimp_sweep.sh <number of experiments> <number of iterations>

Here input the required number of experiments and number of iterations to be simulated, in place of <number of experiments> & <number of iterations>
